# Бот-генератор контента для НКО

Умный Telegram-бот для автоматизированного создания контента для некоммерческих организаций. Генерирует посты для соцсетей, контент-планы и предоставляет инструменты для редактирования с доработкой через ИИ.

### Основной функционал
- **Управление НКО**: Создание и выбор организаций с описаниями и примерами постов
- **Умная генерация контента**: Создание текстов для соцсетей с помощью ИИ
- **Планирование контента**: Стратегические контент-планы с учетом ресурсов
- **Интерактивное редактирование**: Полный цикл правок с доработкой через ИИ
- **Поддержка платформ**: Адаптированный контент для Телеграма и ВКонтакте

### Типы контента
- **Создание постов**: Анонсы событий, отчеты, сбор средств, информационные кампании
- **Контент-планы**: Стратегическое планирование с учетом видео-ресурсов
- **Темы для постов**: Генерация идей и тем для публикаций

## Технологии

- **Python 3.8+** с асинхронным программированием
- **Aiogram 3.x** - современный фреймворк для Telegram ботов
- **SQLite** - база данных для хранения НКО и контента
- **Finite State Machine (FSM)** - управление сложными диалогами
- **AI API** - интеграция с моделями генерации текста
